Este programa resuelve un problema mediante Busqueda en Espacio de Estados
especificamente mediante el tipo de anchura iterativo. Se selecciono 
anchura iterativo puesto que con el te aseguras que la solucion dada sea 
la de el menor tiempo posible, si se hubiese usado profundidad, puede
que requiera menos estados para llegar el final, pero puede llegar a
soluciones en las que el tiempo total no sea el mas optimo.

El programa al abrirlo cuenta con un menu donde solo admite 2 opciones
numericas, 1 para iniciar la ejecucion y 0 para salir del programa. 
Cualquier otro tipo de entrada no sera valido.

Luego contara con la segunda opcion, la cual correspondera a si desea
utilizar los nombres por defecto (opcion 2) o si desea cambiar cada
nombre de los integrantes del grupo (opcion 1), cualquier otro tipo de entrada
no sera valida.

Luego vendran 5 preguntas, correspondientes al tiempo maximo que tiene
el grupo para cruzar, el cual el usuario debe ingresar solamente
numeros enteros positivos, cualquier otro tipo de entrada sera invalida
y se le volvera a pedir al usuario que ingrese una entrada valida. Ocurre 
lo mismo para el tiempo de cada integrante del grupo.

Finalmente, si se encuentra un estado final, los pasos para llegar a este
se muestran por pantalla y de no ser asi, se informa que con los datos
ingresados no es posible encontrar una solucion.

Finalmente se le vuelve a preguntar al usuario si desea ingresar la 
informacion de los integrantes o si desea salir del programa, es decir,
se vuelve al menu.
